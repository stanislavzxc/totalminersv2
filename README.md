# TotalMiners Backend v2 вӣҸпёҸ

Р‘СҚРәРөРҪРҙ-СҮР°СҒСӮСҢ РҝР»Р°СӮС„РҫСҖРјСӢ **totalminers.io** вҖ” СҒРҫРІСҖРөРјРөРҪРҪРҫРіРҫ РјР°Р№РҪРёРҪРі-РҫСӮРөР»СҸ. РЎРөСҖРІРёСҒ Р°РІСӮРҫРјР°СӮРёР·РёСҖСғРөСӮ СғСҮРөСӮ РјРҫСүРҪРҫСҒСӮРөР№, СғРҝСҖР°РІР»РөРҪРёРө РәР»РёРөРҪСӮСҒРәРёРјРё СғСҒСӮСҖРҫР№СҒСӮРІР°РјРё Рё С„РёРҪР°РҪСҒРҫРІСӢРө РҝРҫСӮРҫРәРё.

## рҹӣ пёҸ РўРөС…РҪРҫР»РҫРіРёСҮРөСҒРәРёР№ СҒСӮРөРә
* **Frameworks:** Python (Flask / FastAPI) вҖ” РіРёРұСҖРёРҙРҪР°СҸ Р°СҖС…РёСӮРөРәСӮСғСҖР° РҙР»СҸ РіРёРұРәРҫР№ РҫРұСҖР°РұРҫСӮРәРё СҚРҪРҙРҝРҫРёРҪСӮРҫРІ Рё РІСӢСҒРҫРәРҫРҝСҖРҫРёР·РІРҫРҙРёСӮРөР»СҢРҪСӢС… Р°СҒРёРҪС…СҖРҫРҪРҪСӢС… Р·Р°РҙР°СҮ.
* **Database:** PostgreSQL (РҙР»СҸ РҪР°РҙРөР¶РҪРҫРіРҫ С…СҖР°РҪРөРҪРёСҸ РҙР°РҪРҪСӢС…) + Alembic (РҙР»СҸ СғРҝСҖР°РІР»РөРҪРёСҸ РјРёРіСҖР°СҶРёСҸРјРё).
* **Containerization:** Docker & Docker Compose (РҝРҫР»РҪРҫРө РҫРәСҖСғР¶РөРҪРёРө РІ РәРҫРҪСӮРөР№РҪРөСҖР°С…).
* **Task Management:** Celery / РҗСҒРёРҪС…СҖРҫРҪРҪСӢРө Р·Р°РҙР°СҮРё (СҒРәСҖРёРҝСӮСӢ Р°РІСӮРҫРјР°СӮРёР·Р°СҶРёРё).

## рҹ”Ң РҳРҪСӮРөРіСҖР°СҶРёРё
1. **Headframe API** вҖ” СҒРёРҪС…СҖРҫРҪРёР·Р°СҶРёСҸ РҙР°РҪРҪСӢС… РҝСғР»Р°, РҫСӮСҒР»РөР¶РёРІР°РҪРёРө С…РөСҲСҖРөР№СӮР° Рё СҒСӮР°СӮСғСҒР° РІРҫСҖРәРөСҖРҫРІ РјР°Р№РҪРёРҪРі-РҫСӮРөР»СҸ РІ СҖРөР°Р»СҢРҪРҫРј РІСҖРөРјРөРҪРё.
2. **Bybit API** вҖ” Р°РІСӮРҫРјР°СӮРёР·Р°СҶРёСҸ С„РёРҪР°РҪСҒРҫРІСӢС… СҲР»СҺР·РҫРІ, РҫСӮСҒР»РөР¶РёРІР°РҪРёРө РәСғСҖСҒРҫРІ РәСҖРёРҝСӮРҫРІР°Р»СҺСӮ, РҫРұСҖР°РұРҫСӮРәР° РІРҪСғСӮСҖРөРҪРҪРёС… СӮСҖР°РҪР·Р°РәСҶРёР№ Рё РҪР°СҮРёСҒР»РөРҪРёР№.

## рҹҡҖ Р‘СӢСҒСӮСҖСӢР№ Р·Р°РҝСғСҒРә (Deployment)

> **вҡ пёҸ Р’Р°Р¶РҪРҫ:** РқР° СӮРөРәСғСүРёР№ РјРҫРјРөРҪСӮ Р°РІСӮРҫРјР°СӮРёСҮРөСҒРәРёРө РјРёРіСҖР°СҶРёРё РұР°Р·СӢ РҙР°РҪРҪСӢС… РјРҫРіСғСӮ СҖР°РұРҫСӮР°СӮСҢ РҪРөСҒСӮР°РұРёР»СҢРҪРҫ. Р”Р»СҸ РәРҫСҖСҖРөРәСӮРҪРҫР№ РёРҪРёСҶРёР°Р»РёР·Р°СҶРёРё СҒС…РөРјСӢ Р‘Р” РёСҒРҝРҫР»СҢР·СғР№СӮРө РҝСҖРҫСҶРөРҙСғСҖСғ СҖСғСҮРҪРҫРіРҫ РІРҫСҒСҒСӮР°РҪРҫРІР»РөРҪРёСҸ РҪРёР¶Рө.

### 1. РҹРҫРҙРіРҫСӮРҫРІРәР° РҫРәСҖСғР¶РөРҪРёСҸ
РЎРҫР·РҙР°Р№СӮРө С„Р°Р№Р» `.env` РІ РәРҫСҖРҪРөРІРҫР№ РҙРёСҖРөРәСӮРҫСҖРёРё РҝРҫ РҝСҖРёРјРөСҖСғ `..env.swp` Рё Р·Р°РҝРҫР»РҪРёСӮРө РҪРөРҫРұС…РҫРҙРёРјСӢРө РҙРҫСҒСӮСғРҝСӢ (Р‘Р”, API РәР»СҺСҮРё Headframe/Bybit).

### 2. РҳРҪРёСҶРёР°Р»РёР·Р°СҶРёСҸ РұР°Р·СӢ РҙР°РҪРҪСӢС… Рё Р·Р°РҝСғСҒРә
Р Р°Р·РІРөСҖРҪРёСӮРө РәРҫРҪСӮРөР№РҪРөСҖ РұР°Р·СӢ РҙР°РҪРҪСӢС…, РҝСҖРёРҪСғРҙРёСӮРөР»СҢРҪРҫ РҪР°РәР°СӮРёСӮРө СҒСӮСҖСғРәСӮСғСҖСғ РёР· SQL-РҙР°РјРҝР°, Р° Р·Р°СӮРөРј Р·Р°РҝСғСҒСӮРёСӮРө РҫСҒСӮР°Р»СҢРҪСӢРө СҒРөСҖРІРёСҒСӢ:

```bash
# 1. Р—Р°РҝСғСҒРәР°РөРј СӮРҫР»СҢРәРҫ РәРҫРҪСӮРөР№РҪРөСҖ СҒ PostgreSQL
sudo docker compose up -d db

# 2. РҳРјРҝРҫСҖСӮРёСҖСғРөРј СҒСӮСҖСғРәСӮСғСҖСғ СҒС…РөРјСӢ РІ СҖСғСҮРҪРҫРј СҖРөР¶РёРјРө
psql -U dbuser -h localhost -p 5432 totalminersdb < init_schema.sql

# 3. РҹРҫРҙРҪРёРјР°РөРј РІСҒРө РҫСҒСӮР°Р»СҢРҪСӢРө СҒРөСҖРІРёСҒСӢ РұСҚРәРөРҪРҙР° (API, Admin, Tasks)
sudo docker compose up -d
```
